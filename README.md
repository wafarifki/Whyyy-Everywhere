**Whyyy Everywhere** adalah ekstensi browser untuk mengganti seluruh gambar di website apa pun dengan gambar pejabat terwhyyy Indonesia.

Internet terlalu normal? Aktifkan **Whyyy Everywhere**, lalu biarkan semua gambar di website berubah menjadi versi yang lebih tidak terduga, lebih absurd, dan jauh lebih **WHYYY**.

Ekstensi ini cocok untuk hiburan, prank ringan, konten meme, dan eksperimen kecil-kecilan tentang bagaimana rasanya menjelajahi internet kalau semua gambarnya tiba-tiba berubah menjadi pejabat terwhyyy Indonesia.

## Deskripsi Singkat

Ekstensi browser untuk mengganti seluruh gambar di website apa pun dengan gambar pejabat terwhyyy Indonesia.
Tested:
- Chrome
- Ms Edge

## Fitur

- Mengganti seluruh gambar pada halaman website.
- Mendukung elemen `<img>`.
- Mendukung `srcset`.
- Mendukung `<picture>` dan `<source>`.
- Mendukung lazy-loaded image seperti `data-src`, `data-srcset`, dan atribut sejenisnya.
- Mengganti CSS `background-image: url(...)`.
- Tersedia beberapa pilihan pejabat **WHYYY**.
- Tersedia All Mode untuk menggunakan semua gambar secara acak.
- Memiliki tombol ON / OFF.
- Tidak melakukan request gambar berulang-ulang ke URL eksternal.
- Popup dibuat simple, modern, dan mudah digunakan.

## Mode yang Tersedia

- ### Prabowo Mode
Mengubah seluruh gambar website menjadi gambar Prabowo.

- ### Jokowi Mode
Mengubah seluruh gambar website menjadi gambar Jokowi.

- ### Megawati Mode
Mengubah seluruh gambar website menjadi gambar Megawati.

- ### Soeharto Mode
Mengubah seluruh gambar website menjadi gambar Soeharto.

- ### Semua Mode
Menggunakan semua gambar yang tersedia secara acak. Satu gambar bisa menjadi Prabowo, gambar lain bisa menjadi Jokowi dan seterusnya. Hasilnya biasanya tidak masuk akal. Dan justru itu letak asiknya.

## Cara Download

Ada dua cara untuk menggunakan project ini.

### 1. Download ZIP dari GitHub

1. Buka halaman repository project ini di GitHub.
2. Klik tombol **Code**.
3. Pilih **Download ZIP**.
4. Extract file ZIP yang sudah didownload ke folder baru klean

### 2. Download dari halaman release Github

1. Buka halaman release di project ini / buka link dibawah
```text
https://github.com/wafarifki/Whyyy-Everywhere/releases
```
2. Pastikan download versi terbaru jika ada, lalu klik **Source code (zip)**
3. Extract file ZIP yang sudah didownload ke folder baru klean

## Cara Install di Google Chrome

Karena ekstensi ini belum dipublish ke Chrome Web Store, klean bisa menginstallnya secara manual melalui Developer Mode.

### 1. Buka Halaman Extensions

Buka Google Chrome, lalu ketik:

```text
chrome://extensions
```
di tab bar Google Chrome, lalu klik enter

### 2. Aktifkan Developer Mode

Di pojok kanan atas halaman Extensions, aktifkan toggle:

```text
Developer mode
```

### 3. Load Extension

Klik tombol:

```text
Load unpacked
```

Lalu pilih folder project:

```text
whyyy-everywhere
```

Pastikan yang dipilih adalah folder yang berisi file project ini.

### 4. Pin Extension ke Toolbar

Agar lebih mudah digunakan:

1. Klik ikon puzzle/extensions di toolbar Chrome.
2. Cari ikon **Whyyy Everywhere**.
3. Klik ikon untuk nge pin ekstensi.
4. Ekstensi akan muncul di toolbar Chrome.

## Cara Menggunakan

### Mengaktifkan Ekstensi

1. Buka website apa pun.
2. Klik ikon ekstensi **Whyyy Everywhere** di toolbar Chrome untuk membuka popup **Whyyy Everywhere**
3. Pilih mode yang ingin digunakan.
4. Ubah status menjadi **ON**.
5. Gambar di website akan berubah sesuai mode yang dipilih.

### Mematikan Ekstensi

2. Klik ikon ekstensi **Whyyy Everywhere** di toolbar Chrome untuk membuka popup **Whyyy Everywhere**
2. Ubah status menjadi **OFF**.
3. Website akan kembali normal setelah halaman dimuat ulang otomatis.

### Mengganti Mode

Untuk mengganti mode:

2. Klik ikon ekstensi **Whyyy Everywhere** di toolbar Chrome untuk membuka popup **Whyyy Everywhere**
2. Pilih mode lain, misalnya dari Prabowo ke Jokowi.
3. Website akan dimuat ulang otomatis.
4. Gambar akan berubah sesuai mode dengan pilihan terbaru.

## Catatan Penggunaan

Ekstensi juga mungkin tidak bekerja sempurna di beberapa website yang menggunakan proteksi gambar, shadow DOM tertutup, canvas rendering, atau mekanisme rendering khusus.

## Kenapa Namanya Whyyy Everywhere?

Karena setelah ekstensi ini aktif, reaksi paling masuk akal biasanya adalah:

```text
Why?
```

Lalu setelah lihat seluruh website berubah total:

```text
Whyyy?
```

Dan karena efeknya muncul di mana-mana, jadilah:

```text
Whyyy Everywhere
```
